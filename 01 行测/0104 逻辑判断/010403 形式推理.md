# 1.假言命题 - 通过集合来理解

A           ->      B

充分条件            必要条件

## 充分条件表述

如果...那么...

若...则...

只要...就...

一旦...就...

...就...

...必须...

...一定...

所有...是...

...是...

## 必要条件表述

只有...才...

...是...的前提

...是...的基础

...是...不可或缺的

...是...的先决条件

不...不...

没有...就没有...

必须...才...

除非...否则非(不)...

## 充要条件表述

如果A那么B；只有A才B

A当且仅当B

A是B的唯一要求

A是B的唯一条件

## 假言命题推理 (利用上述推理，对选项进行筛选)

前提：A -> B 为真

规则：

(1) 肯前必肯后

(2) 否后必否前

(3) 否前未必否后

(4) 肯后未必肯前

## 假言命题矛盾

A -> B 

等价命题  -B -> -A  

矛盾命题  A ∩ -B

## 假言命题串联

A -> B, B-> C, 则 A -> C

规则：

传递性：肯前必肯后

逆否性：否后必否前

## 假言命题归谬

(1) A -> B, -A -> B, 则 B 为真

(2) A -> B, A -> -B, 则 A 为假

# 2.联言命题

A ∩ B , 指事件A和事件B同时发生

## 联言命题的表述

A 和 B

A 且 B

A 但是 B

不但 A, 而且 B

虽然 A, 但是 B

既 A 又 B

一边 A 一边 B

## 联言命题真假

有假 -> 假

全真 -> 真

另外补充：一定真 -> 可能真

## 联言命题矛盾

-(A 且 B) = -A 或 -B

## 联言命题总结

(1)A且B, 指事件A和事件B同时发生
(2)A真；B真 -> A且B 真
(3)A且B 真 -> A 真；
   A且B 真 -> B 真；
   A且B 真 -> A真 且 B真；
(4)A且B 假 -> A假 或者 B假

# 选言命题

## 相容性选言命题

A 或者 B，是指事件A和事件B至少发生一件，也可以都发生

## 相容性选言命题表述

A 或者B

或者A 或者B

也许A 也许B

可能A 可能B

不是A 就是B

A和B至少一个

或者A，或者B，二者至少其一

## 相容性选言命题真假

有真 -> 真

全假 -> 假

## 相容性选言命题推理

真 -> 有 真

假 -> 全假

## 相容性选言命题矛盾

-(A或B) = -A 且 -B

## 相容性选言命题等价

原始： “A 或 B 为真” 等价于 “-A -> B”   等价于“-B -> A”

矛盾： 以上三个说法的矛盾命题均为：-A且-B

PS:

A -> B 的矛盾命题 A且-B
A -> B 的等价命题 -A或B

喜哥说：所有的假言命题都是选言命题

## 不相容性选言命题含义

要么A要么B，它的含义是事件A和事件B发生且仅发生一个

## 不相容性选言命题表述

要么A，要么B

或者A或者B，二者必居其一

或者A或者B，二者不可得兼

## 不相容性选言命题真假

一真一假 -> 真

全真全假 -> 假

## 不相容性选言命题推理

真 -> 一真一假

假 -> 全真全假

## 不相容性选言命题矛盾

-(要么A要么B) = 要么(A且B) 要么(-A且-B)
